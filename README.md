## ğŸ” What This System Does

Upload an image of a structure (wall, beam, column, slab, etc.) and the system automatically generates a **detailed structural inspection report**, including:

- Identification of structural defects (cracks, spalling, corrosion, exposed reinforcement, etc.)
- Classification of each defect
- Severity assessment (Low / Medium / High)
- Potential structural and safety impact
- Estimated time before permanent damage
- Short-term and long-term repair solutions
- Approximate repair cost (INR) and implementation time
- Preventive measures to avoid future defects
- Engineer-style formatted report using bullets and tables
